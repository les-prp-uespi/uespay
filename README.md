# UesPay

Sistema de carteira digital universitária para modernização do acesso e pagamento de serviços dentro da universidade, com foco inicial no Restaurante Universitário (RU) da UESPI.

---

## Sobre o Projeto

O **UesPay** é uma plataforma digital desenvolvida para facilitar pagamentos de serviços universitários utilizando créditos digitais. O projeto busca substituir métodos tradicionais, como fichas físicas e pagamentos presenciais, por uma solução moderna, prática e segura.

O MVP (Produto Mínimo Viável) tem como objetivo inicial permitir:

- Consulta de saldo;
- Histórico de transações;
- Pagamento digital;
- Leitura e geração de QR Code;
